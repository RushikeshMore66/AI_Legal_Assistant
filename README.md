An intelligent Indian Legal AI Assistant that leverages Retrieval-Augmented Generation (RAG), Groq LLM, and HuggingFace embeddings to deliver accurate and context-aware legal information. 

The system understands user queries, maps them to relevant laws (IPC, Constitution, etc.), explains legal concepts in simple language, and provides actionable guidance. It also includes conversation memory and case tracking to simulate real lawyer-like interactions.
